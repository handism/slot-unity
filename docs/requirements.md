# 要件定義書 — Fantasy Slot

**バージョン**: 1.1
**作成日**: 2026-03-20
**最終更新日**: 2026-03-28（Issue #58: ベット選択 UI 仕様を実装に合わせて更新）
**対象プラットフォーム**: PC (Windows / macOS)
**エンジン**: Unity 6.3 LTS

---

## 1. ゲーム概要

ファンタジーをテーマにした 5 リールのアーケードスロットゲーム。
プレイヤーはコインを消費してリールをスピンし、ペイライン上にシンボルを揃えることで配当を得る。
フリースピンおよびボーナスラウンドを備え、長時間の遊戯に耐えうるゲームループを提供する。

---

## 2. 機能要件

### 2.1 リール・ペイライン

**注**: 本ドキュメント内でのリール番号は「1始まり（1〜5）」で表記しますが、プログラム内のインデックスは「0始まり（0〜4）」として扱います。

| 項目 | 仕様 |
|------|------|
| リール数 | 5 |
| 表示行数 | 3 行（各リールに 3 シンボルを表示） |
| ペイライン数 | 25 ライン（固定） |
| 判定方向 | 左から右（左端リールのシンボルから連続一致） |
| 最小一致数 | 3 シンボル以上で配当発生 |

**25 ペイラインのレイアウト（行を Top=0, Mid=1, Bot=2 で表現）:**

```
Line 01: Mid  Mid  Mid  Mid  Mid   (─────)
Line 02: Top  Top  Top  Top  Top   (─────)
Line 03: Bot  Bot  Bot  Bot  Bot   (─────)
Line 04: Top  Mid  Bot  Mid  Top   (∧∨)
Line 05: Bot  Mid  Top  Mid  Bot   (∨∧)
Line 06: Mid  Top  Top  Top  Mid
Line 07: Mid  Bot  Bot  Bot  Mid
Line 08: Top  Top  Mid  Bot  Bot
Line 09: Bot  Bot  Mid  Top  Top
Line 10: Top  Mid  Mid  Mid  Bot
Line 11: Bot  Mid  Mid  Mid  Top
Line 12: Mid  Mid  Top  Mid  Mid
Line 13: Mid  Mid  Bot  Mid  Mid
Line 14: Top  Top  Bot  Top  Top
Line 15: Bot  Bot  Top  Bot  Bot
Line 16: Mid  Top  Mid  Bot  Mid
Line 17: Mid  Bot  Mid  Top  Mid
Line 18: Top  Bot  Top  Bot  Top
Line 19: Bot  Top  Bot  Top  Bot
Line 20: Top  Mid  Top  Mid  Top
Line 21: Bot  Mid  Bot  Mid  Bot
Line 22: Top  Bot  Bot  Bot  Top
Line 23: Bot  Top  Top  Top  Bot
Line 24: Mid  Mid  Top  Top  Top
Line 25: Mid  Mid  Bot  Bot  Bot
```

### 2.2 シンボル仕様

| ID | 名前 | 種別 | 3揃え | 4揃え | 5揃え |
|----|------|------|------:|------:|------:|
| 0 | Dragon（竜） | 高配当 | ×10 | ×25 | ×125 |
| 1 | Phoenix（鳳凰） | 高配当 | ×7 | ×15 | ×75 |
| 2 | Crystal（魔法石） | 高配当 | ×5 | ×10 | ×50 |
| 3 | Sword（聖剣） | 高配当 | ×3 | ×6 | ×30 |
| 4 | Ace | 低配当 | ×2 | ×4 | ×12 |
| 5 | King | 低配当 | ×1 | ×3 | ×10 |
| 6 | Queen | 低配当 | ×1 | ×2 | ×8 |
| 7 | Jack | 低配当 | ×1 | ×2 | ×5 |
| 8 | Wild（魔法使い） | 特殊 | — | — | — |
| 9 | Scatter（魔法陣） | 特殊 | — | — | — |
| 10 | Bonus（宝箱） | 特殊 | — | — | — |
| 11 | Blank（空白） | フィラー | — | — | — |

- **Wild**: 特殊シンボルを除く任意シンボルの代替として機能。Scatter にはならない。
- **Scatter**: リール上どこに出現しても判定。3個以上でフリースピン発動。配当倍率表あり（後述）。
- **Blank（フィラー）**: ペイラインを遮断するだけで配当なし。各リールストリップ 88 スロット中 26 個（約 30%）配置し、有効シンボルの出現率を下げることで RTP を目標値（約 96%）に調整するための確率調整用シンボル。
- **配当額**: 「ベット額 × 倍率」で算出。

**Scatter 配当:**

| 個数 | 配当倍率 |
|------|------:|
| 3個 | ×2 |
| 4個 | ×10 |
| 5個 | ×50 |

### 2.3 コイン・ベット

| 項目 | 仕様 |
|------|------|
| 初期コイン | 1,000 枚 |
| ベット選択肢 | 10 / 20 / 50 / 100 コイン/スピン |
| デフォルトベット | 10 コイン |
| コイン上限 | 9,999,999 枚（表示上限） |
| ゲームオーバー条件 | コインが 0 になりスピン不可になった時 |
| ゲームオーバー後 | コインをデフォルト値（1,000枚）にリセットして続行 |

### 2.4 ボーナス機能

#### 2.4.1 フリースピン

| 項目 | 仕様 |
|------|------|
| 発動条件 | Scatter 3個以上 |
| スピン回数 | 3個→10回 / 4個→15回 / 5個→20回 |
| 配当倍率 | 通常スピン配当の ×2 |
| コスト | 無料（コイン消費なし） |
| 再トリガー | フリースピン中に Scatter 3個以上で追加付与（上限 +20回） |
| ベット額 | 発動時のベット額を引き継ぐ |

#### 2.4.2 ボーナスラウンド

| 項目 | 仕様 |
|------|------|
| 発動条件 | 同一スピンでリール 1・3・5（インデックス 0, 2, 4）すべてに特定のボーナスシンボル（Scatter と共用しない専用シンボル扱い）が出現 ※実装上は Scatter 5個とは別条件 |
| 内容 | 宝箱選択ミニゲーム。9個の宝箱から3個選択。各宝箱に隠れたコイン報酬を獲得 |
| 報酬範囲 | ベット額の ×5 〜 ×100（ランダム、RTP に基づいて重み付け） |
| ハズレ箱 | なし（必ずコイン獲得） |

> **注**: ボーナスラウンドとフリースピンが同時発動した場合、ボーナスラウンドを優先し、終了後にフリースピンへ移行する。

### 2.5 RTP（理論還元率）

| スピン種別 | 実現 RTP |
|-----------|------:|
| 通常スピン | 約 74.5% |
| フリースピン・ボーナス込み全体 | 約 96.0% |

リールの重み設定（Blank シンボル 26 個配置）により、高精度で目標 RTP を実現。
詳細は `RTP_Result.csv` を参照。

### 2.7 UI 要件

| 要素 | 説明 |
|------|------|
| タイトル画面 | ゲーム起動後のエントランス画面。「START」ボタン押下でメインゲームへ遷移。ロゴ・ボタン・背景の動的演出（ブレス演出、回転、浮遊）を含む。 |
| リール表示 | 5×3 グリッド。スピン中はアニメーション |
| コイン残高 | 常時表示。獲得時はカウントアップ演出 |
| ベット選択 | 10 / 20 / 50 / 100 コインの 4 段階から選択する専用ボタンでベット額を変更。選択中のベットはハイライト表示（金色グラデーション）。 |
| スピンボタン | 押下でスピン開始。スピン中は停止ボタンに変化（早期停止可能） |
| オートスピン | 10 / 25 / 50 / 100 回の自動スピン設定。残高不足・ボーナス発動で自動停止。選択肢ボタンはランタイムに動的生成される（`MainHUDView.autoSpinCounts` 配列で管理）。選択肢を変更する場合は `MainHUDView.cs` の `autoSpinCounts` フィールドを Inspector または直接編集すること |
| ペイライン表示 | 当選ライン強調表示（非当選ラインはフェード） |
| 当選演出 | 当選額ポップアップ + シンボルアニメーション |
| フリースピン表示 | 残り回数・累計獲得コインを画面上部に常時表示 |
| ボーナスラウンド画面 | 専用 UI（宝箱選択画面）へ遷移 |
| 設定メニュー | BGM / SE ボリューム調整、ゲーム説明、コイン残高リセット |
| ペイテーブル | 各シンボルの配当一覧を参照できるモーダル |

---

### 2.8 サウンド要件

| 種別 | 内容 |
|------|------|
| BGM | 通常プレイ用（ループ）、フリースピン用（ループ）、ボーナスラウンド用（ループ） |
| SE | スピン開始、リール停止（×5 順次）、当選（小/中/大）、ビッグウィン、Scatter 出現、フリースピン発動、ボーナスラウンド発動、宝箱選択・開封、ボタン操作 |
| ボリューム | BGM / SE 独立してスライダー調整可能（0〜100%）。設定値はローカル保存 |

### 2.9 データ永続化

| データ | 保存方式 | タイミング |
|--------|---------|-----------|
| コイン残高 | JSON ファイル（Application.persistentDataPath） | スピン結果確定後 |

### 2.10 ターボモード

| 項目 | 仕様 |
|------|------|
| 概要 | スピンからリール停止までの演出時間を短縮し、より高速にゲームを進行できるモード |

## 3. 非機能要件

### 3.1 パフォーマンス

| 項目 | 目標値 |
|------|--------|
| フレームレート | 60 FPS（通常時） |
| 起動時間 | 5 秒以内（中スペック PC） |
| メモリ使用量 | 512 MB 以下 |

### 3.2 品質

- コインが 0 になった状態でスピンできないこと
- フリースピン中にオートスピン中断が正常に動作すること
- セーブデータが破損した場合にデフォルト値でフォールバックすること
- ゲームウィンドウのリサイズ・フルスクリーン切り替えで UI レイアウトが崩れないこと（16:9 基準、レターボックス対応）

### 3.3 保守性

- シンボル追加・ペイライン変更が ScriptableObject の編集のみで対応できること
- ボーナス条件・配当倍率がコード変更なしに調整できること

---

## 4. 対象外（スコープ外）

- サーバー連携・オンランキング
- 課金・マネタイズ機能
- モバイル対応（iOS / Android）
- 多言語対応

## 5. 用語集

| 用語 | 定義 |
|------|------|
| リール | 縦スクロールするシンボル列 |
| ペイライン | 配当判定の経路 |
| ベット | 1スピンに消費するコイン枚数 |
| 配当 | ベット × 倍率で計算されるコイン獲得量 |
| RTP | Return to Player。理論上の払い戻し率 |
| Wild | 特殊シンボルを除く代替シンボル |
| Scatter | 出現位置を問わず判定される特殊シンボル |
| フリースピン | コイン消費なしの無料スピン |
| ボーナスラウンド | 宝箱選択のミニゲーム |
